# Desi-Heritage-Food-Validator-

"A Python project that uses custom exceptions to filter traditional Indian dishes from modern fast food"

## 📖 Introduction

The Indian Heritage Food Validator is a specialized tool dedicated to preserving the authenticity of India’s vast and ancient culinary landscape. From the spice-heavy kitchens of the North to the coastal delicacies of the South, this project ensures that traditional recipes are honored and validated against true heritage standards.

In a world where Indian cuisine is often generalized, this validator acts as a digital safeguard for the specific techniques, ingredients, and soul that define authentic Indian cooking.

## 🎯 Objectives

* **Validate Authenticity:** Use Python logic to verify traditional Indian ingredients.
* **Filter Fast Food:** Deploy custom exceptions to catch and reject modern "fusion" errors.
* **Map Origins:** Ensure dishes align with their specific regional roots (e.g., Punjabi, South Indian).
* **Code for Culture:** Demonstrate how technical error-handling can preserve heritage data.

## 🛠️ How it Works (Methodology)

* **Step 1:** I created a list of "Rules" for authentic Indian food.
* **Step 2:** I used **Python** to write the validation logic.
* **Step 3:** I made **Custom Exceptions** to stop the code if it finds "wrong" ingredients like processed cheese or mayo.
* **Step 4:** The program checks if the ingredients match the specific region of the dish.
