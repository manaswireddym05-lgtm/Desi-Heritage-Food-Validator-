# Desi-Heritage-Food-Validator-

"A Python project that uses custom exceptions to filter traditional Indian dishes from modern fast food"

## 📖 Introduction

The Desi Heritage Food Validator is a Python-based tool designed to map Indian states to their traditional culinary identities. In an era of global fast-food fusion, this project acts as a digital safeguard to ensure that when a user searches for a region, they are presented with an authentic heritage dish that defines that state's culture.

## 🎯 Objectives

Regional Mapping: To create a reliable link between Indian states and their iconic heritage foods.
Data Validation: To implement Python logic that handles user input (case-sensitivity and spacing) to ensure accurate results.
Cultural Awareness: To educate users on the traditional food heritage of India through a simple, interactive terminal interface.
Scalable Code: To build a foundation that can later be expanded into a full culinary encyclopedia.

* **Validate Authenticity:** Use Python logic to verify traditional Indian dishes.

* **Map Origins:** Ensure dishes align with their specific regional roots (e.g., Punjabi, South Indian).

## 🛠️ How it Works (Methodology)

* **Step 1:** I created a list of "Rules" for authentic Indian food.
* **Step 2:** I used **Python** to write the validation logic.
* **Step 3:** I made **Custom Exceptions** to stop the code if it finds "wrong" ingredients like processed cheese or mayo.
* **Step 4:** The program checks if the dish match the specific state of india.
